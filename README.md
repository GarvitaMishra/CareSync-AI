# CareSync-AI

AI-powered healthcare platform for smart hospital discovery, booking management, analytics, payments, refunds, and healthcare administration.

# 🚀 Features

## 👤 User Features
- Secure JWT Authentication
- Hospital & service search
- Smart AI-based recommendations
- Distance-based hospital discovery
- Slot booking system
- Razorpay payment integration
- Booking cancellation & refund tracking
- Booking history dashboard
- Interactive chatbot assistant

## 🏥 Hospital Owner Features
- Owner dashboard
- Hospital management
- Service management
- Booking monitoring
- Analytics overview
- Responsive admin-style UI

## 📊 Admin Features
- Analytics dashboard
- Revenue tracking
- Most popular services
- Top-performing hospitals
- Platform activity monitoring

# 🧠 AI Features
- AI hospital scoring system

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/CareSync-AI.git
```

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

## Frontend Setup

```bash
cd frontend
npm install
npm start
```
